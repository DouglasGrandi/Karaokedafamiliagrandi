
# Karaokedafamiliagrandi (Android TV/Box)

App de karaokê com **duas opções de reprodução**:
- **YouTube** (via app oficial – Intent)
- **Vídeos locais** (arquivos .mp4/.mkv do seu pendrive/armazenamento) usando **ExoPlayer**

Também aceita **vários links do YouTube de uma vez** (cole um por linha).

## Build rápido
```bash
./gradlew assembleDebug
# APK em: app/build/outputs/apk/debug/app-debug.apk
```

## Instalar
Baixe o APK (Actions → Artifacts) e instale na TV Box (ativar Fontes desconhecidas).

## Observação legal
Não baixe conteúdo do YouTube dentro do app. Use Intent do app oficial.
